# Modern Japanese-Inspired 3D Calculator

This project is a web-based calculator with a modern Japanese-inspired design and 3D effects. Built with Vue, TypeScript, and Vite, it aims to provide a minimalistic and elegant user experience, featuring soft shadows, subtle gradients, and a calming color palette inspired by Japanese Zen aesthetics.

### Features

- **Japanese-inspired Design**: Minimalistic UI with a neutral color palette and soft shadows.
- **3D Effects**: Subtle gradients and shadows for a lifted, tactile feel.
- **Basic Calculator Functions**: Perform basic arithmetic operations with clear and functional buttons.
- **Responsive Layout**: Adjusts to different screen sizes.

### Technologies Used

- **Vue 3**: JavaScript framework for building UI.
- **TypeScript**: Strongly typed JavaScript for improved development experience.
- **Vite**: Fast build tool for modern front-end development.
